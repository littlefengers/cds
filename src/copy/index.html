Utilities

Utilities are simple HTML classes typically scoped to a single CSS property like border-style or background-color. Utilities can be used additively to style an object from scratch or to override a style defined in component CSS.

Colors

Utility Mixin Example CSS 
tbd
 
@include cds-u-color-bg($color)
 
@include cds-u-color-bg("costco-blue")
 
background-color: #005dab
 
tbd
 
@include cds-u-color-border($color)
 
@include cds-u-color-border("costco-blue")
 
border-color: #005dab
 
tbd
 
@include cds-u-color-border($color)
 
@include cds-u-color-text("costco-blue")
 
color: #005dab
